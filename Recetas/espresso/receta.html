Espresso

Ingredientes: 1- 7 gramos de café, molienda para espresso. 2- 50 ml de agua. 3- Cafetera espresso.

Procedimiento: 1- Se prepara la cafetera, para eso se siguen las instrucciones del fabricante. 2- Se coloca el café molido en el filtro, y se compacta con un tamper. 3- Se inserta el filtro en la base y se comprueba la temperatura (de ser posible), la indicada está entre 85 y 90 grados. Se inicia la extracción. 4- Se esperan de 25 a 30 segundos para cerrar el agua, en ese tiempo se debe haber obtenido un total de 25 mililitros de un café oscuro, con sabor y aroma fuerte y una capa mediana de crema sobre él. 5- Se limpia el filtro y la cafetera.

¡Y listo!
